Privacy Policy

This Privacy Policy describes how CNX Connect collects and uses information when you use the application.

Application Information

App Name: CNX Connect

Developer: Satyam Tiwari

Contact Email: it@castnx.com

Information We Collect

The application may collect the following information:

  • Device IMEI number
  • Login credentials (IMEI and password)
  • Device operational data received from the connected VFD device

How We Use the Information

The collected information is used only for:

  • Connecting the mobile application with the VFD device
  • Displaying device parameters and statistics
  • Monitoring device faults and operational status
  • Providing real-time device information to the user

Location Information

This application does not collect the user's personal location. The location displayed inside the application refers only to the connected device location, not the user's device location.

Internet Usage

The application requires internet connectivity to communicate with the developer's secure server and retrieve device data.

Third-Party Services

The application does not use third-party analytics or advertising services. All data communication happens only with the developer's own server.

Advertisements

This application does not display advertisements.

Data Security

We take appropriate security measures to protect user and device information from unauthorized access.

Changes to This Privacy Policy

This Privacy Policy may be updated in the future. Any changes will be reflected on this page.
